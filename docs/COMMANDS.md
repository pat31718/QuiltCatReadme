# 棉被貓 QuiltCat — 指令列表

> 本文件列出棉被貓目前所有可用指令，依照指令類別與權限等級分類。  
> **前綴**：預設為 `cat`。

---

## 指令權限分類說明

| 標記 | 說明 |
|------|------|
| （無標記） | 一般指令，所有使用者皆可使用 |
| 🔒 | 管理員指令，需要伺服器擁有者或管理員身分 |

---

## 可愛互動指令

| 指令 | 說明 |
|------|------|
| `cat meow` | 棉被貓發出「喵~」的叫聲 |
| `cat pat` | 摸摸棉被貓 |

---

## 金錢指令

| 指令 | 說明 |
|------|------|
| `cat money` | 顯示自己持有的小魚乾（現金 + 存款） |
| `cat money @使用者` | 查看指定使用者的小魚乾 |
| `cat money @使用者 +/-<數量>` | 🔒 增加或減少使用者的小魚乾現金 |
| `cat money give @使用者 <數量>` | 將自己的小魚乾送給其他使用者 |
| `cat daily` | 每日簽到，獲得小魚乾獎勵（連續簽到有加成） |
| `cat daily @使用者 <欄位> <值>` | 🔒 修改使用者簽到資料 |
| `cat work` | 工作賺取小魚乾（有 30 分鐘冷卻時間） |
| `cat work time` | 查看工作冷卻剩餘時間 |
| `cat work force` | 🔒 強制執行工作（忽略冷卻時間） |

---

## 銀行指令

| 指令 | 說明 |
|------|------|
| `cat bank` | 顯示銀行資訊（現金、存款、利率、利息倒數） |
| `cat bank +<數量>` | 存款（將現金存入銀行） |
| `cat bank -<數量>` | 提款（從銀行提出現金，需扣除手續費） |
| `cat bank @使用者 +/-<數量>` | 🔒 增減使用者的小魚乾存款 |
| `cat bank modify X%` | 🔒 將所有人存款修改為 X% |
| `cat bank Interest` | 🔒 強制立即發放利息 |

---

## 寵物指令

### 一般寵物指令

| 指令 | 說明 |
|------|------|
| `cat pet` | 顯示自己的寵物資訊（若尚未領養，會自動引導領養） |
| `cat pet new` | 領養一隻新寵物 |
| `cat pet upgrade` | 進化寵物（需滿足天數、屬性點數、小魚乾條件） |
| `cat feed` | 餵食寵物（消耗背包中的鳴戶卷，收納盒中的寵物也會一併餵食） |
| `cat pet collect` | 查看寵物圖鑑（顯示已收集與未收集的寵物種類、收集進度） |
| `cat pet release` | 放生寵物（免費重置，記錄到圖鑑後放生，需 yes/no 二次確認） |
| `cat pet reset` | 等級回退（消耗水果，將寵物退回上一等級，存活時間與屬性點數重置為零） |
| `cat pet box` | 查看寵物收納盒 |
| `cat pet swap <編號>` | 交換跟隨中的寵物與收納盒中指定編號的寵物 |
| `cat pet @使用者` | 偷看其他使用者的寵物 |

### 管理員寵物指令

| 指令 | 說明 |
|------|------|
| `cat pet @使用者 <屬性> <值>` | 🔒 修改使用者的寵物數值 |
| `cat pet @使用者 collect clear` | 🔒 清空使用者的寵物圖鑑 |
| `cat pet @使用者 collect addall` | 🔒 填滿使用者的寵物圖鑑（所有可收集種類） |
| `cat pet @使用者 collect <寵物種類>` | 🔒 新增指定寵物種類到使用者的圖鑑 |
| `cat pet @使用者 box unlock` | 🔒 解鎖或擴充使用者的收納盒欄位 |

> 可修改的寵物屬性：`petType`、`petName`、`life`、`food`、`intPoint`、`dexPoint`、`strPoint`、`survivalTime`、`feedTimes`

| `cat use <道具名稱>` | 使用 1 個道具 |

## 優先度高的說明指令

以下指令會在主要指令的回應中以提示文字顯示，引導使用者使用：

| 主指令 | 提示的子指令 | 條件 |
|--------|-------------|------|
| `cat pet` | `cat feed` | 寵物存活時 |
| `cat pet` | `cat pet upgrade` | 滿足升級條件時 |
| `cat pet` | `cat pet new` | 寵物死亡（飽食度為 0）時 |
| `cat feed` | `cat pet new` | 寵物死亡時 |
| `cat pet new` | `cat pet swap` | 寵物移入收納盒時 |
| `cat pet release` | `cat pet new` | 放生後無其他寵物時 |
| `cat pet box` | `cat pet swap <編號>` | 收納盒有寵物時 |
| `cat pet box` | `cat pet new` | 收納盒為空時 |
