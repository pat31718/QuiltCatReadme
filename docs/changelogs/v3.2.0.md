# 🐱 QuiltCat Beta 3.2.0 版本更新公告

> 發佈日期：2026-04-12

## 🆕 新增功能

### 寵物圖鑑系統
- 新增 `cat pet collect` 指令，可查看寵物圖鑑
- 圖鑑依照等級分組顯示，已收集顯示 ✅、未收集顯示 ❓
- 顯示收集進度（已收集數 / 總數）
- 領養寵物、進化、等級回退時自動記錄至圖鑑

### 寵物放生功能
- 新增 `cat pet release` 指令，可免費放生目前跟隨中的寵物
- 需回答 `yes` 或 `no` 作為二次確認，防止誤操作
- 若收納盒中有寵物，放生後會自動將收納盒中第一隻寵物設為跟隨

### 寵物等級回退功能
- 新增 `cat pet reset` 指令，可將寵物等級回退一階
- 回退需消耗對應屬性的水果（檸檬 / 葡萄 / 櫻桃）
  - LV1 → Basic：5 顆水果
  - LV2 → LV1：20 顆水果
  - LV3 → LV2：50 顆水果
- 回退後存活時間與屬性能力點數將重置為零

### 寵物收納盒系統
- 新增 `cat pet box` 指令，可查看收納盒中的寵物
- 新增 `cat pet swap <編號>` 指令，可將收納盒中的寵物與跟隨中的寵物交換
- 當任一寵物達到最高等級（目前為 LV2）時，自動解鎖 1 個收納盒欄位
- `cat pet new` 指令更新：若目前有存活的寵物且收納盒有空位，會自動將現有寵物移入收納盒

## 🔧 改善與修正

### 餵食功能更新
- `cat feed` 指令更新：餵食後會同時餵養收納盒中的所有寵物
- 嵌入訊息中會分別顯示每隻寵物的餵食數量

### 錯誤修正
- 修正自動餵食迴圈中誤用 `MoneyData.TIMER_ITEM_ID`（道具 4，神秘的打卡鐘）的問題，現已正確使用 `PetData.FEED_ITEM_ID`（道具 2，鳴戶卷）

## 📦 資料遷移

本次更新變更了寵物資料的儲存結構（`PetData` → `PetBox`），系統會在首次讀取時自動將舊格式資料遷移為新格式，整個過程對使用者來說是無感的，不需要手動操作。
